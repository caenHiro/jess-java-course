# Java Course — Jess

Repositorio de aprendizaje Java — 90 días desde cero hasta Spring Boot.

## Estructura

```
├── vault/                    # Notas en Obsidian
│   ├── 00_Guia_Uso.md       # Empieza aquí
│   ├── 01_Instalacion/      # Guías de instalación (Java, VS Code, Git, Docker, Maven, Spring)
│   ├── 02_Teoria/           # Teoría semanal
│   ├── 03_Practicas/        # Ejercicios (tú los resuelves aquí)
│   └── 04_Notas_Personales/ # Tus notas personales y progreso
└── codigo/                   # Tu código Java
    ├── semana-01/
    ├── semana-02/
    └── ...
```

## Cómo empezar

1. Instala las herramientas: `vault/01_Instalacion/00_Stack_del_Curso.md`
2. Lee la guía de uso: `vault/00_Guia_Uso.md`
3. Empieza la semana 1: `vault/02_Teoria/semana-01.md`

## Subir avances

```bash
bash scripts/push.sh "semana-01 primer programa"
```

## Actualizar (cuando hay nuevo contenido)

```bash
git pull
```
